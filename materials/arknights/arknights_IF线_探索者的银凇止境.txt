因非冰原，萨米存在自身的意志，接近于一种自然法则。邪魔（坍缩体）从北方持续入侵，萨
米深处矗立着传说中的“星门”。本集成战略包含两种独立玩法模块：【常规行动】（三结局）

// ★★★ 常规行动三结局 ★★★

// 一结局：越过群山（正常通关）

??     萨米肉鸽-一结局-越过群山       萨米因非冰原深处   麦哲伦、博士、凯尔希、哥伦比亚科考队   无特殊条
件通关第五层即可解锁。这是萨米肉鸽常规行动的标准结局、默认世界线。莱茵生命科考队在
萨米北部山地与自称独眼巨人的萨卡兹先知建立合作关系。探险家麦哲伦深入萨米地底发现象
征萨米心脏的巨型植物结构“安玛的心脏”，并破译记载当地痛楚的密文，科考队遭遇坍缩体侵
袭后持续北进并完成了探索目标。最终抵达冰原的尽头——“星门”。结局文本《越过群山》收
录于冬夜展览馆边缘历史集。萨米的意志得到了尊重，探索成功，人类取得了对邪魔作战的关
键知识，但深层威胁未完全解除。

// 二结局：坠渊已久（接取邪魔之力）

??     萨米肉鸽-二结局-坠渊已久       萨米/邪魔污染领域  博士、玛利图斯、邪魔             达成条件：获得一定的
“坍缩值”，在特定不期而遇中选择接触并利用邪魔的力量。泰拉人在对抗邪魔的过程中未能抵
御侵蚀，主动或被动地接受了邪魔之力，导致心智扭曲或力量失控，成为了邪魔入侵的通道或
污染源。代表着“人类被邪魔腐化”的IF结局。

// 三结局：与山交谈（邪魔完全入侵）

??     萨米肉鸽-三结局-与山交谈       萨米冰原尽头的星门   麦哲伦、博士、星门             达成条件：达成一定探索
条件并击败最终BOSS“萨米的意志”后解锁。萨米最悲观与绝望的结局分支。星门被成功打开，
但其连接的另一端是更为古老和不可名状的存在。邪魔彻底入侵现实世界，整个泰拉大陆陷入
认知混乱与坍缩风险，萨米的战士们只能靠着守护家园的本能战斗，冰原与星门的秘密向无穷
的黑暗延伸。

通过最终的险路恶敌战斗后自动达成。“错误的推演，错误的责任，错误的牺牲。你为后来者
