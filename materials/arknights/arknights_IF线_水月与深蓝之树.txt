// 前置信息

??     水月与深蓝之树-故事背景       伊比利亚/深海     博士、水月、斯卡蒂、凯尔希         深海线IF集
成战略，以海嗣个体水月与博士的生存抗争为主线。海洋已经沉寂了太久，待到它苏醒并发声
之时，万籁都将归于寂静。改变流向的因素太多，从盐风城的提问到愚人号的警告，从猎人们
的殊死搏杀到离群者的流浪路途。只要一步走错，潮水就会在顷刻之间淹没我们脚下的土地。
本集成战略设有四个结局。

// 一结局：平凡即是喜乐

??     水月肉鸽-一结局-平凡即是喜乐  盐风城/伊比利亚   水月、海沫、博士、凯尔希         不满足其他结局
解锁条件的情况下，常规通关第五层（击败BOSS“屠谕者”即大群意志）。这是默认结局。在
另一条世界线中，阿戈尔提前介入阻止灾难，深海猎人海沫获救，在博士与水月的努力下被带
回了罗德岛。剧情相对平和，代表了一种相对平和的收束，从屠谕者的手中夺回了海沫。水月
在此线中保持了“人”的身份，没有成为海嗣。

// 二结局：静谧时代

??     水月肉鸽-二结局-静谧时代      深海大静谧后的废墟   水月、斯卡蒂、凯尔希           解锁条件：前三层不期而遇“开端”
选择“跟随大群”（获得藏品“主教的研究”，灯火-5），击杀“骑士”后（需将骑士保护至最后一
层），五层关底BOSS替换为“最后的骑士”，击败后达成。第二次大静谧席卷了陆地。斯卡蒂
成为海嗣之神“伊莎玛拉”并带领海嗣飞升宇宙。凯尔希虽守住了人类最后的城市，成为“最后一
座城市的守护者”，但水月为保护博士而牺牲了自己。人类文明基本宣告终结。二结局也是后
续月度小队的核心前置故事线。

// 三结局：息潮的代价

??     水月肉鸽-三结局-息潮的代价    深海核心           水月、伊莎玛拉(斯卡蒂)、博士      解锁条件：结局2通关后，三
层前不期而遇“开端”选择“决定潜入深海”获得“主教的研究”；后续事件“深蓝之心”选择“搜寻园
林”获得“深蓝之心”（灯火-50，1名干员立刻出现排异反应）；解锁隐藏第六层，击败BOSS“伊
莎玛拉”后达成。水月为从根源上平息海嗣的同化本能，牺牲自己融入名为“始源的命脉”的大群
意识。他以自身的意志压制了伊莎玛拉，自身的意识也溶解于深海中，只有博士还记着水月与
他的约定。剧情结尾为开放式：伊莎玛拉被平息，但水月彻底消失了。

// 四结局：深蓝之星（Stella Caerulea）

??     水月肉鸽-四结局-深蓝之星      外太空/宇宙         水月、伊祖米克、斯卡蒂          水月与“深蓝之树”融合，
成为新生的海嗣之神“伊祖米克”。他带领海嗣族群进化并飞升宇宙，泰拉危机在另一种层面被
化解。斯卡蒂在此线中的命运有多种解读：有的理解为斯卡蒂也随之飞升，也有解读认为最终
留在博士身边的只是继承了斯卡蒂之形貌的海嗣之神。无论哪种理解，水月/伊祖米克最终回
到了博士身边。

// 【补充】水月肉鸽月度小队剧情IF前置

??     水月肉鸽-月度小队·遗孤们      深海/罗德岛        斯卡蒂、幽灵鲨、W           二结局“静谧时代”是后续所
有月度小队的IF前置线。在二结局伊莎玛拉胜利后必然发生的结果下，斯卡蒂成为“浊心”，月
度小队“遗孤们”涉及深海猎人最后的挣扎，斯卡蒂、幽灵鲨等角色在二结局前提下的命运走向。
