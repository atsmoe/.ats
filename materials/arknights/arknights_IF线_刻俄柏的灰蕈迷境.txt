// 前置信息

??     刻俄柏的灰蕈迷境-故事背景      幻梦境/刻俄柏的内心世界   刻俄柏(小刻)                 集成战略首个年度主
题，刻俄柏吃下“灰蕈”（致幻蘑菇）后进入的幻梦旅程。灰蕈迷境并非仅仅是一个简单的迷幻
梦，更是触及泰拉源石秘密、大地方舟底层的“精神视界”。

// 一结局：迷尘幻梦

??     灰蕈肉鸽-一结局-迷尘幻梦       幻梦境            刻俄柏、浮士德(幻影)             达成条件：任意难度通
关第五层，关底BOSS固定为法术浮士德。剧情简介：刻俄柏的默认梦境结局。她吃下灰蕈后经
历一场光怪陆离的冒险，最终梦境结束，她被罗德岛的伙伴唤醒。一切仿佛只是一场奇异而不
真实的梦，“就像做了一场大梦”，在罗德岛的宿舍中醒来。

// 二结局：茫然行者

??     灰蕈肉鸽-二结局-茫然行者       幻梦境            刻俄柏、寒灾(幻影/霜星镜像)      达成条件：通关普通难
度结局一后，完成“不存在的童话1”选择花5原石下葬；“不存在的童话2”花费三次目标点获“迷
幻的藏宝图”；“不存在的童话3”获“染血的传单”，第五层BOSS替换为“寒灾”，击败后达成。
剧情简介：刻俄柏在幻境中迷失，未能成功走出迷途。她独自走在荒野之上，看到的“寒灾”
（形象与霜星相似的敌人）成为她心中无法跨越的冰霜。刻俄柏在梦境中不断徘徊，茫然行走
着，这是一种不完全的迷失，没有彻底死亡但也无法回归清醒。

// 三结局：归于荒野

??     灰蕈肉鸽-三结局-归于荒野       幻梦境第六层/源石深处   刻俄柏                      达成条件：通关结局二
后，“死斗之后”获“沉重的铁皮箱”；第三层BOSS战取得“漆黑的钥匙”；“灾难将至”获“手枪”
（收藏品NO.168）；击败第五层法术浮士德后进入隐藏第六层，击败BOSS“墓碑”后解锁。剧
情简介：最隐秘的结局。刻俄柏未能走出幻境，她的灵魂与意识消散，“化作泥土，永远地活
在了这片大地上”。隐藏结局暗示刻俄柏触碰到了“这片大地真相的残片”，灰蕈迷雾渗入了她
的灵魂。所谓“大地的真相”很可能指向源石的本质——源石是有自我意识的，刻俄柏无意中发
现这个真相，这令她体内的源石迅速做出反应，“在幻境中杀死了她”。多种解读：刻俄柏与源
石的深层意识融合，成为了某种“王座”或是被大地所吞噬。
